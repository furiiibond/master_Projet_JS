Puissance 4

Binvenue dans le jeux de puissance 4, vous devez jouer contre votre adversaire. Vous allez jouer chaquns votre tour en plaçant des pions dans les cases vides. Le but du jeu est de placer des pions dans la colonne de gauche à droite. Le premier joueur à aligner 4 pions dans une ligne, colonne ou diagonale gagne. Le jeu se termine aussi lorsqu'un joueur a rempli la grille.

Le jeu est réalisé en Javascript et utilise les canvas. Les pions sont représentés par des cercles.

Pour jouer appuiez sur la touche de votre clavier correspondant à la colonne.

Differentes action supplémentaires sont possibles:
  • - presser le bouton "RAZ" pour remettre à zéro la grille.
  • - presser le bouton "Sauver" pour sauvegarder la grille.
  • - presser le bouton "Resor" pour restaurer la grille.
  • - choisir une couleur de depart.
  • - presser le bouton "?" pour afficher les indications.
